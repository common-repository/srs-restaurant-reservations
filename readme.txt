A simple reservations management plugin for your Restaurant.

== Description ==

SRS Reservations allows easy management of your restaurant reservations. Our main focus is at the back-end dashboard that gives you a quick overview for the selected day.

= Features =

* Allows customers to make reservations through your website
* Add reservation form to any page using simple short code
* Graph shows the number of reservation against their times allowing to figure out the busiest time in advance that helps in planning ahead
* Reservation categories (pending, confirmed, deleted)
* Confirm, edit or delete reservations
* List all the reservations for the selected date or date range
* Easily create reservation for customers who call or walk-in
* Settings allow you to customise opening times, reservation duration and maximum number of people for a reservation
* Email confirmation (future versions)
* Responsive design to support mobile devices for the back-end dashboard (future versions).

Reservations can be made through front-end "the customer fills the reservation form" or by the staff members using the back-end. The Dashboard allows staff members to easily create reservations for customers who call or walk-in.

The Dashboard gives a quick overview for the selected date like how many reservations you have, how many are still pending and needs confirmation. It also allows to edit, confirm, delete any reservation. Graph helps you figure out your busy and quite times that should help you prepare accordingly.

Settings page allows configure the opening and closing times, number of maximum people per reservation and reservation's maximum duration.

= Shortcode =
Add [srs_reservations] to any post or page to add the reservation form

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'SRS Restaurant Reservations'
3. Activate 'SRS Restaurant Reservations' from your Plugins page.
4. Browse to SRS Restaurant Reservations dashboard by clicking on Reservations on left menu.
5. Start adding reservations :)

= From WordPress.org =

1. Download 'SRS Restaurant Reservations'.
2. Upload the 'SRS Restaurant Reservations' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate SRS Restaurant Reservations from your Plugins page. 
4. Browse to SRS Restaurant Reservations dashboard by clicking on Reservations on left menu.
5. Start adding reservations :)
